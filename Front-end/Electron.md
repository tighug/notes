# Electron

Electronは、**クロスプラットフォームのデスクトップアプリを開発するためのフレームワーク**です。ChroniumとNode.jsを利用しているため、ブラウザアプリと同じ技術でデスクトップアプリが開発可能です。

## Installation

```bash
yarn add electron
```

## CLI

```bash
electron [main file]  # Run Electron App
```
